# LaTeX

## Math

* `\alpha`, `\Alpha`
* `\beta`, `\Beta`
* …
* `\sin`, `\cos`, `\tan`, …

Subscripts — `a_1` `a_{n+1}`

Superscripts — `a^2` `a^{n+1}`

`\frac{1}{2}`

`\sqrt{2}`

### Sums

* `\sum_{x=1}^{\infty} \frac{1}{x}`
* `\sum_{init}^{limit} term`

### Integrals

`\int_{lower}^{upper} term \mathrm{d}x`

### Big operators

* `\sum`
* `\prod`
* `\int`

* Roman — `\mathrm`

* Bold — `\mathbf`
* Calligraphic — `\mathcal`
* Double-lined — `\mathbb`

* `\exists`, `\nexists`, `\forall`
